
# Docker Workshop

A hands-on workshop for learning Docker containerization and deployment practices.

## Prerequisites

- Docker installed and running
- Basic command-line knowledge
- Familiarity with Git

## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/Sohila-Khaled-Abbas/docker-workshop.git
cd docker-workshop
```

2. Explore the workshop materials and follow along with each module.

## Workshop Modules

- **Introduction to Docker**: Core concepts and architecture
- **Building Images**: Creating Dockerfiles and custom images
- **Running Containers**: Managing containers and their lifecycle
- **Networking**: Container communication and networking
- **Volumes and Storage**: Persistent data management
- **Docker Compose**: Multi-container applications

## Common Commands

```bash
docker build -t image-name .
docker run -d --name container-name image-name
docker ps
docker logs container-name
docker stop container-name
```
